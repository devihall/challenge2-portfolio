Latest Project

Work Scheduler

A daily a simple calendar application that allows a user to save events for each hour of the day. This app runs in the browser and feature dynamically updated HTML and CSS, powered by jQuery and utilizes the Bootstrap framework. Users can open the planner to see current day is displayed at the top of the calendar. Users can scroll down to see time blocks for hours of the day. Each time block is color-coded to indicate whether it is in the past, present, or future. Users can click into a time block to enter an event. Users can click the save button for that time block, then that event can be saved in local storage. When the page is refreshed the saved events persist in local storage. GitHub Link, and Workscheduler App.

Technologies used include:

  • HTML
  • CSS
  • jQuery
  • Bootstrap

Employee-Tracker

A content management systems (CMS) based on the command-line, to manage a company's employee database. Built using Node.js, Inquirer, and MySQL. The MySQL2 package connects to the SQL database and performs queries, the Inquirer package interacts with the user via the command line, and the console.table package prints MySQL rows to the console. Personal Project

Smart Webcam

A smart webcam with TensorFlow.js pretrained machine learning model. A webpage that uses machine learning directly in the web browser via TensorFlow.js to classify and detect common objects, (yes, including more than one at a time), from a live webcam stream. This can supercharge your regular webcam to identify objects and get the coordinates of the bounding box for each object it finds, and highlights the found object in the video stream. Based on a Google Codelabs guided tutorial. GitHub and Smart webcam page
